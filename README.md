# simple-twitter-frontend
專案網址：https://tzynwang.github.io/simple-twitter-frontend/

![screenshot](./.README/before-login.png)

## 功能介紹
- 可在首頁瀏覽全部推文
- 可在任意頁面開啟新增推文之對話框，執行新增推文
- 在大多數畫面皆支援執行回覆推文、對推文按讚的行為
- 使用者可修改自身名稱、封面圖片、大頭照、自我介紹，以及帳號相關設定
